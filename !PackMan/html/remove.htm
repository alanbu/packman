Remove Dialog

The Remove Dialog allows you to confirm or cancel the removal of a package.

It is shown when you hit the toolbar Remove button or select the menu Remove option.

It shows the following information

  • Name of the package to remove
  • The version that will be removed
  • The description summary of the package
  • List of other packages that will be removed because they depend on this package.
  • Total number of packages to be removed

      • Press the Remove button to remove this package and packages that depend on it. This will then display the Commit Window which will show the progress of the removal and report if it is successful or not.

      Press the Cancel button if you do not want to remove this package.
