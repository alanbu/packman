Iconbar icon

Click on the iconbar with the select button to see the main window that is used for maintaining the packages.

Click on the iconbar with the menu button to show a menu with the following choices:
Info Shows information on the PackMan application including the current version
Help Launches the Help file to show help on PackMan
Advanced Leads to the Advanced menu that provides advanced options for configuring the packaging system.
Upgrade All Upgrades all installed packages which have an upgrade available. Either the Upgrade All dialogue is shown that lists the packages that will be upgraded and lets you upgrade them all in one go, or a message is shown if there are no upgrades currently available. If there are no upgrades available you may want to run the update lists option and try again to see if any new upgrades have appeared since the last time you updated the lists.
Update Lists This downloads the latest versions of the package lists that are used to provide the list of packages that are available. It shows a window to indicate the progress of the List update. This Window also contains an "Upgrade All" button that is enabled after the download of the lists if there are any upgrades available.
Quit Close the package Manager
