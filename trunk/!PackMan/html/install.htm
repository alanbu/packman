Install Dialog

The Install Dialog allows you to confirm or cancel the installation/upgrade of a package.

It is shown when you hit the toolbar Install button or select the menu Install option.

It shows the following information

  • Name of the package to install/upgrade
  • The version that will be installed
  • The description summary of the package
  • List of other packages that will be installed automatically as they are need for the correct running of this package
  • List of packages that need to be removed. Usually because they have been replaced by this package or one of its dependencies or conflict with this package
  • Totals of the number of packages to be installed and removed
  • Total size of the downloads required to install this package

      • Press the Install button to install this package and its dependencies and remove any clashing packages. This will then display the Commit Window which will show the progress of the installation and report if it is successful or not.

      Press the Cancel button if you do not want to go ahead and install or upgrade this package.
