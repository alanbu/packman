Installing

  • Drag the !PackMan application out of the zip archive onto a temporary location to run it from
  • Double click on !PackMan to run the application
  • Selected Update Packages from the Iconbar icon menu.
  • If this is the first time you've run !PackMan (and you haven't used !RiscPkg before) you will be presented with a dialog box asking you to create the !Packages directory. It is recommended you accept the default location and press OK. Once the !Packages directory is installed. Select Update Packages again.
  • A window will appear showing the progress of downloading the package lists. When it has finished click OK to close the Window.
  • Click on the icon bar to show the Main Window which allows you to install and remove packages

After trying out !PackMan, if you wish to continue using it, please install the packman package from the main window. This will allow !PackMan to be upgraded the same as any other package.

Once !PackMan has been installed in this way it is placed in Apps.Admin. You should delete the version you downloaded first and use this new version from now on
