Search Dialog

The search dialog allows you to search for the given text in a packages name or description.

The results are shown in the main window and the Filter in the toolbar is changed to "Search Results filter". The window will be empty if no packages contain the text.

The fields on the dialog are:

  • Search for - Enter the text to be searched for
  • Search in current filter - set this check box to restrict the search to the packages showing in the current filter
  • Cancel - Close the dialog without searching.
  • Search - Do the package search
