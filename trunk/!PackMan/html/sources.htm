Sources Window

The Sources Window allows you to specify the locations where packages are held.

You may have multiple package sources defined at once and they will be merged into a single list in PackMan.

  • Add... Will show a dialog allowing an new source to be added
  • Edit... Select an existing source with the mouse and click this button to show a dialog allow a source to be edited. Double clicking on an existing source will also show the edit dialog
  • Remove Select an existing source with the mouse and click this button to remove it
  • Refresh package lists - select this check box to automatically run the Update Lists command when the sources are saved to download and use the latest versions of the lists.
  • Cancel - Cancel the dialog and discard any changes
  • Save - Save the new list of sources.

At the time of writing this manual the sources I was aware of were

  • http://www.riscpkg.org/pkg/Unstable - This is the main packageing list. It is recommended that you always have this source in your list.
  • http://www.riscos.info/packages/pkg/autobuilt - This contains a useful list of ported applications, games, libraries and the latest GCC compiler.
  • http://www.netsurf-browser.org/downloads/development/riscpkg/packages - This contains the latest development build of the NetSurf browser.
