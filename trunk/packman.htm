PackMan is a utility to help install and maintain applications and other components of a RISC OS computer.

This is version 0.2 (first public Beta version). It should be feature complete, but may contain some rough edges and bugs. Please send me feedback to improve this application. Contact details are in the application Help.

This is an alternative frontend for packages in the RISC OS Packaging Project format.

If you are already using !RiscPkg there is a PackMan package you can install.

If you are not using !RiscPkg, you will need a program that can decompress Zip files (e.g. !SparkPlug) and to have installed the SharedUnixLibrary which is available from here.
