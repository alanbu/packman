Application Menu

The application menu gives options for manipulating the currently selected package

This menu is a sub menu shown from the menu for the main window

It provide the following options

  • Install - install or upgrade the selected application. When selected it leads to the Remove Dialog to remove the package. This option is disabled if the application is not installed.
  • Apps - shows the Applications Window which shows the applications installed by this package and allows them to be linked to and copied. It is disabled if the application is not installed.
  • Info - shows the Information window that gives more information on the package.
  • Copyright - shows the copyright information for the package. It is disabled if the application is not installed.
