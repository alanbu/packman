Commit Window

The Commit Window shows the progress of an installation, upgrade or removal of a package and it's dependencies.

It displays a message of what stage in the process it is in and a progress bar to show how far through the operation it is.

If the operation completes successfully. Click on the button to dismiss it.

If the operation fails an error window will be shown with the cause of the failure.

If the failure is caused because there are already files in the location it would like to install files in one of the installed packages the Files that conflict Window listing these files will also be shown.
