Toolbar

The toolbar runs across the top of the main menu and provides icons to give short cuts to common features of PackMan and the filter that allow you to show a subset of the packages.

The icons are faded if they are not currently available. e.g. The first four need an item to be selected.

The items on the toolbar are (from left to right):

  • Install/Update icon - Installs or Updates the selected package. A green arrow on the icon pointing right is an install, an orange one pointing up is an upgrade. Selecting on the icon shows the Install window
  • Remove icon - Removes the selected package. Selecting the icon shows the Remove window
  • Apps icon - Shows the Applications Window which shows you what application are in an installed package and allows you to view their location and create links to them.
  • Info icon - Show information about the package in the Package information window
  • Filter list - select the arrow by the side of the current filter and it shows a menu, which allows you to filter the packages displayed by their status (installed, upgrades), the search result or a category of package.
  • Search icon - show the Search Window that allows the package names text to be searched.
